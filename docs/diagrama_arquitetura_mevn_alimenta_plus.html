Diagrama visual de alto nível com fluxo de ações e atores do sistema (Stack MEVN)

Fluxo do Doador (Ricardo)

  • Ação 1: Ricardo faz login e aceita o Termo de Segurança Jurídica na interface Vue.js.
  • Ação 2: Ao final do expediente, preenche o formulário expresso com a sobra de comida.
  • Ação 3: O Vue envia as coordenadas e os dados estruturados via Axios ao Express, que salva instantaneamente no MongoDB como uma doação de status "DISPONIVEL".

Fluxo do Receptor (Sandra)

  • Ação 1: Sandra abre o painel da ONG em Vue.js para analisar o feed de alimentos ativos.
  • Ação 2: Filtra a listagem por bairros ou categorias específicas de interesse da cozinha.
  • Ação 3: Clica em "Reservar". O sistema altera o status do documento para "RESERVADO" no MongoDB, bloqueando o lote para outras instituições e disponibilizando as orientações de acesso ao Bruno.

Inteligência de Integração

  • ViaCEP: Quando qualquer usuário digita seu CEP, o Express atua de forma síncrona batendo na API externa para preencher a rua e o bairro, garantindo a exatidão logística que o voluntário de campo precisa para a retirada física.
  • Mongoose: Garante que mesmo em um banco de dados NoSQL flexível, nenhum pacote de alimento seja criado sem campos obrigatórios (como quantidade ou data limite de expiração).
